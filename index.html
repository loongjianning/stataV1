第一阶段：掌握核心实证方法 (Causal Inference)

深挖雷老师论文中的计量内核，理解政策评估与因果识别机制。

📈双重差分法 (DID) & 事件研究法

  • 应用场景： 政策冲击评估 (如：司法垂直管理改革、4万亿经济刺激)。
  • 核心要求： 必须掌握平行趋势检验 (Parallel Trend) 与动态效应图的绘制 (Event Study)。
  • 关注重点： 政策交乘项设定，控制组的选择逻辑。

🌍工具变量法 (IV)

  • 应用场景： 解决内生性问题，寻找外生冲击。
  • 高阶案例： 利用“全球油价波动”构建针对产油/依赖性地方收入的“Bartik”式外生冲击 (参考：Seeing Corruption Differently)。
  • 检验要求： 第一阶段F值检验、排他性约束论证。

📏断点回归 (RD)

  • 应用场景： 基于硬性行政/人口门槛。
  • 高阶案例： 城市人口是否达到300万对地铁项目审批及市长晋升的影响。
  • 核心检验： 驱动变量 (Running Variable) 的密度检验 (McCrary density test)。

🛡️稳健性检验 (Robustness)

  • 替换变量： 使用客观数据如夜间灯光亮度代替GDP。
  • 安慰剂检验 (Placebo)： 随机打乱政策时间节点或处理组身份，绘制T值/系数分布图。

第二阶段：进阶 Stata 编程技术与数据清洗

处理中国城市面板数据、官员政治升迁数据，掌握现代高维固定效应运算。

1. 高维固定效应与聚类标准误 (High-Dimensional FE)

现代实证标配命令 reghdfe。常用于处理包含城市(City)、年份(Year)甚至省份-年份交乘(Province*Year)固定效应的模型。

// 标准高维固定效应模型及聚类标准误调整 reghdfe Y X1 X2 controls, absorb(citycode year i.provcode#c.year) vce(cluster citycode)

2. 宏微观数据匹配 (Data Merging)

将《中国城市统计年鉴》的宏观特征与 CCER Elite Database（官员背景库）匹配。

// 处理官员换届与城市年度数据的不对齐问题 use "city_macro.dta", clear // 使用城市代码和年份进行匹配 merge 1:m citycode year using "ccer_officials.dta" keep if _merge == 3 drop _merge

IV (工具变量)

Seeing Corruption Differently

专家与公众腐败感知差异的信息学视角

实证亮点： 利用国际油价的剧烈波动作为宏观经济波动的“外生冲击”，结合面板数据，论证专家评分系统如何被非制度性因素(如经济好坏)扭曲。

技术落点： 截面与面板IV回归分析，绘制相关性图表验证第一阶段逻辑。

DID (双重差分)

Making Local Courts Work

司法垂直管理改革与地方保护主义

实证亮点： 巧妙利用中国司法系统控制权逐步向上集权(recentralization)的改革节点。

技术落点： 搜集 2012-2018 上市公司诉讼微观数据，构建 DID 模型，并控制了大量高维固定效应以剔除干扰。

TWFE / Database

The Political Resource Blessing...

庇护网络、基础设施投资与中国经济发展

实证亮点： 深度依赖 CCER Elite Database 追踪官员庇护网络。处理复杂的宏观基础设施面板数据。

技术落点： 使用 reghdfe (Correia, 2016) 执行双向固定效应估计，通过精细的数据清洗识别政治网络。
